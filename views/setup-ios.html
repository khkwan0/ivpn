Manual Setup

Set up your VPN on iPhone (iOS)

Open Settings

On your iOS device, tap Settings > General > VPN (scroll down) > Add VPN configuration > Setup VPN configuration

Select L2TP on the top, then enter the following settings:

  • Description: Enter any description (for example: My VPN).
  • Server: Type the desired IgniteVPN server address. (Or use 23.253.99.109)
  • Account: Type your username you registered with.
  • RSA SecurID: Make sure it is off.
  • Password: Type your password you registered with.
  • Secret: Type engagebdr
  • Send All Traffic: Make sure it is on.
Tap Save in the top-right corner.
