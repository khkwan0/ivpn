Manual Setup

Set up your VPN on a PC

Open Settings

To add a new VPN connection, click the Internet access icon in your system tray. From the menu that pops up, select Network settings.

In the Network & Internet settings panel, select VPN > Add a VPN connection and enter the details below.

  • VPN provider: Windows (built-in)
  • Connection name: Make up a name for your connection.
  • Server name or address: Type the desired IgniteVPN server address. (Or use 23.253.99.109)
  • VPN type: L2TP over IPSec
  • Type of sign-in info: User name and password
  • User name: Type your username you registered with.
  • Password: Type your password you registered with.
Check the box for Remember my sign-in info and then click Save.

Connect to the VPN server location

You will now see your new VPN connection in the Network & Internet settings panel. To connect to the VPN server, click the name of the connection, then click Connect.


Congratulations! You can now begin surfing with freedom and security! Open your web browser, and all traffic will automatically be protected by IgniteVPN.
