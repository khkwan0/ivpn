Manual Setup

Set up your VPN on Android

Open Settings

Go to Settings > More... > Wireless & Networks (for other versions of Android, this might be under Settings > Connections > More connection settings instead).

Configure

Select VPN and tap the + on the top right corner (other versions of Android might name this Add VPN instead). On the Edit VPN profile screen, enter the following:


  • Name: Type any name that will help you recognize your VPN connection. You are recommended to type in the location name of the server address you will use later (for example: VPN USA West 1).
  • Type: L2TP/IPSec PSK
  • Server address: Type the desired IgniteVPN server address. (Or use 23.253.99.109)
  • L2TP Secret: (Leave blank)
  • IPSec identifier: (Leave blank)
  • IPSec pre-shared key: Type engagebdr
  • DNS search domains: (Leave blank)
  • DNS servers (e.g.8.8.8.8): (Leave blank)
Tap Save once you are done.

Connect

Tap the VPN profile you just made. Type your Username and Password that you got earlier. Make sure Save account information is ticked. Then tap Connect.


Congratulations! You can now begin surfing with freedom and security! Open your web browser or any app, and all traffic will automatically be protected by IgniteVPN.
